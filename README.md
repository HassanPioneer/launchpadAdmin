# Launchpad Admin Frontend

## Environment

Node version: v16.19.1

## Setup

### Step 1

```bash
yarn
```

### Step 2

Create & update ``.env`` file

```bash
cp .env.example .env
```

### Step 3: Run development server

```bash
yarn start
```

Default port running in ``8666``

You can change in ``.env`` file:

```bash
PORT=8666
```

### Step 4

Access: <http://localhost:8666/#/>
